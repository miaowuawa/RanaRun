# RanaRun

RanaRun乐奈快跑，ComicUP抢票工具

**助力每一场漫展的抢票，实现每一个二次元的梦想！**

### 注意
**为避免黄牛/营利性用户使用本工具，我们永远不会开发，亦不接受以下类型的功能PR：**
- 通过本工具对接其他“验证码接收服务”，”授权平台“，接收验证码，注册comicup账号
- 通过本工具向comicup账号内添加，批量添加实名人，修改实名人
- 自动化的支付宝支付，批量下单，一次性导出所有账号的支付链接等
- 其他任何形式给黄牛/规模化使用本工具者带来便利的功能

### 使用流程（预售）：
1. 注册ComicUP账号
2. 在**自己的手机上**完成账号配置，包括填写实名信息
3. 在本程序上登录账号，挑一个正在售卖的项目**进行一次下单测试**（重要！必做！）
4. 确认测试成功后，预定正式抢票
5. 一旦有票后下单，立即扫描弹出的支付宝二维码

### 使用流程（回流）：
1. 注册ComicUP账号
2. 在**自己的手机上**完成账号配置，包括填写实名信息
3. 在本程序上登录账号，**设置提示音/抢到播放音乐/飞书通知**（重要！）
4. 填写门票信息，以回流模式开始自动化盯门票
5. 一旦有票后自动下单，立即扫描弹出的支付宝二维码
